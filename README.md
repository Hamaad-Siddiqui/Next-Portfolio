# Next-Portfolio

⚡️ My portfolio is inspired by [leerob.io](https://github.com/leerob/leerob.io).
It is a more minimal and a simple version of it.

## Running Locally

```bash
$ git clone https://github.com/Hamaad-Siddiqui/Next-Portfolio.git
$ cd Next-Portfolio
$ yarn
$ yarn dev
```

## Built Using

- [Next.js](https://nextjs.org/)
- [Vercel](https://vercel.com)
- [Tailwind CSS](https://tailwindcss.com/)
